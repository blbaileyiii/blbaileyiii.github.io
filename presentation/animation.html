w3schools.com: CSS @keyframes Rule
It covers most of what I covered, though it doesn't a unique style of animation shown here: 

            
                @keyframes glow {
                    from {
                        text-shadow: 0 0 5px #ffffff00, 0 0 10px #000000dd;
                    }
                    to {
                        text-shadow: 0 0 10px #000000dd, 0 0 20px #000000ff;
                    }
                }

In this example it is a little confusing but the first from statement will transition to the first to statement, then it will go to the second from statement and then the second to statement.

What happens is it transitions from white fully transparent, to black half-transparent, and then continues from black half-transparent to black opaque.

I wanted to point it out since it doesn't really showcase that feature.
